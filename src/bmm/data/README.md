ADEO Business Terms

File: `adeo-business-terms.csv`

Columns: `business_term`, `description`, `definition`

Owner: ADEO Data Architects

Purpose: Canonical glossary of business terms used across ADEO products and platforms. This file is the authoritative external source for terminology. Agents should reference this CSV when validating PRDs, API field names, UI copy, and documentation — but must not embed or duplicate the content in agent configuration or memory.

Update Process: To request a change, contact ADEO Data Architects (data-architects@adeo.example) and provide justification, proposed definition, and impact analysis.
