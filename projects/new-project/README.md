## New project starter

This folder is a safe place to start experimenting inside this repo without touching the main `src/` or build tooling.

### Run

From the repo root:

```bash
node projects/new-project/main.mjs
```

### What to change first

- Edit `main.mjs` to add your logic.
- If you want this to become a real package/app, tell me what kind (CLI, web app, library) and I’ll scaffold it properly.
